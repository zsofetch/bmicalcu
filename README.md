# BMI Calculator 

A modern Body Mass Index (BMI) calculator built with **.NET MAUI**.

## Features:
* Sleek dark-mode aesthetic with high-contrast accents. Very "Brat" by Charlixcx inspired.
* Seamlessly switch between Metric (kg/cm) and Imperial (lb/ft/in).
* Instant category calculation (Underweight, Normal, Overweight, Obese).

## Tech Stack:
* **Framework**: .NET MAUI (2026 Edition)
* **Language**: C# 14 / XAML
* **Platform**: Cross-platform (WinUI, Android SDK 35+)

## How to Run:
1. Clone the repository: `git clone https://github.com/zsofetch/bmicalcu.git`
2. Open `bmicalcu.sln` in **Visual Studio 2022/2026**.
3. Ensure the **.NET MAUI workload** is installed.
4. Press `F5` to debug on your preferred emulator or device.
